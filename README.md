# lab3_501696
Репозиторий для изучения системы контроля версий Git

# Описание
Данная лабораторная работа посвящена изучению системы контроля версий Git.  
В ходе выполнения работы были изучены и применены основные команды для создания, изменения и объединения версий файлов.  
Также была выполнена работа с ветками и разрешение конфликтов при слиянии.

# Цель работы
Освоить базовые приёмы работы с системой контроля версий Git и закрепить практические навыки использования:
- создание и переключение между ветками;
- фиксация изменений с помощью коммитов;
- просмотр различий между версиями;
- выполнение отката и команды rebase;
- разрешение конфликтов при слиянии merge;
- создание итоговой ветки release.

# Выполненные действия
1. Создан репозиторий lab3_repo_501696;
2. В файл config.txt добавлены изменения и выполнены коммиты;
3. Созданы и использованы ветки backend_cuda, feature1 и feature2;
4. Проведены операции rebase и merge с разрешением конфликтов;
5. Создана итоговая ветка release, содержащая все изменения.

## Заключение
В результате выполнения лабораторной работы были изучены основные возможности системы контроля версий Git.  
Полученные знания позволяют эффективно управлять изменениями в проекте, работать с ветками и решать конфликты при объединении кода.
